# Editorial Restyle Design

**Goal:** Restyle the Hugo PaperMod site to match the warm editorial aesthetic of the daily-news-report project.

**Approach:** CSS Override via PaperMod's `assets/css/extended/` mechanism + `layouts/partials/extend_head.html` for Google Fonts. No theme source files modified.

**Key changes:** Warm beige palette, Newsreader serif headings, Outfit sans-serif body, rust accent color, editorial card styling, light-only mode.
